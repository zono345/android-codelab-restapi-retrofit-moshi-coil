# android-codelab-restapi-retrofit-moshi-coil

Introduction
------------
- Android CodeLabで作ったアプリです。
- アプリ名称: MarsPhotos
- インターネットから火星の写真のURLを取得し、画像をダウンロードし表示するアプリです。
  
対象Codelab
